Since then many people have contributed to the project, leading to a rich set of functionality, as given below. == Library == The CDK itself is a library, instead of a user program. However, it has been integrated into various environments to make its functionality available. CDK is currently used in several applications, among which the [[R (programming language)]]<ref>Guha, R. In 2008 bits of GPL-licensed code was removed from the library.
