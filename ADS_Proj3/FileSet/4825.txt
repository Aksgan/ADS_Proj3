A core technology of OneCare was the multi-platform RAV (Reliable Antivirus),<ref>{{cite web |url=http://news.zdnet.co.uk/security/0,1000000189,39286351,00.htm |title=Microsoft: OneCare should not have been rolled out |publisher=CBS Interactive Limited |date={{date|March 16, 2007}} |accessdate={{date|2010-01-29}} }}</ref> which Microsoft purchased from [[GeCAD Software]] Srl in 2003,<ref>{{cite web |url=http://www.microsoft.com/presspass/press/2003/Jun03/06-10GeCadPR.mspx |title=Microsoft to Acquire Antivirus Technology From GeCAD Software: Future Antivirus Solutions, Windows Platform Investments Will Provide Customers With Added Protection |work=Microsoft Presspass |publisher=Microsoft Corporation |date={{date|June 03, 2003}} |accessdate={{date|2010-01-29}} }}</ref> but subsequently discontinued.<ref name="RAV discontinued">{{cite web |url=http://www.infoworld.com/t/applications/microsoft-kill-popular-linux-antivirus-product-443 |title=Microsoft to kill popular Linux antivirus product |publisher=Infoworld, Inc. |date={{date|June 11, 2003}} |accessdate={{date|2010-01-29}} }}</ref> The software has been available as an annual paid subscription, which could be used on up to three computers. On {{date|November 18, 2008}}, Microsoft announced that Windows Live OneCare would be discontinued on {{date|June 30, 2009}} and will instead be offering users a new free anti-malware suite called [[Microsoft Security Essentials]] to be available before then. However, virus definitions and support for OneCare will continue until a subscription expires.<ref>{{cite web |url=http://www.microsoft.com/Presspass/press/2008/nov08/11-18NoCostSecurityPR.mspx |title=Microsoft Announces Plans for No-Cost Consumer Security Offering: New anti-malware solution will broaden PC protection and help improve Windows experience. |work=Microsoft Presspass |publisher=Microsoft Corporation |date={{date|November 18, 2009}} |accessdate={{date|2009-10-10}} }}</ref> In the end-of-life announcement, Microsoft noted that Windows Live OneCare would not be upgraded to work with [[Windows 7]] and would also not work in Windows 7 XP Mode.<ref name="eol">{{cite web |url=http://onecare.live.com/standard/en-US/activation/oc_eol_guidance.htm |title=Windows Live™ OneCare® End of Sale Guidance Page |publisher=Microsoft Corporation |accessdate={{date|2009-11-17}} }}</ref> ==History== Microsoft designed OneCare as part of a [[cloud computing|cloud]]-based computing service named [[Windows Live]] (see [http://ideas.live.com official ideas website]). Windows Live OneCare entered a [[Development stage|beta state]] in the summer of 2005. The managed beta program was launched before the public beta, and was located on BetaPlace, Microsoft's former beta delivery system. On {{date|May 31, 2006}}, Windows Live OneCare made its official debut in retail stores in the [[United States]].<ref>{{cite news |url=http://news.com.com/Microsoft+security+product+makes+official+debut/2100-7355_3-6078249.html |title=Microsoft security product makes official debut |publisher=CNET News.com |last=Evers |first=Joris |date={{date|2006-05-31}} |accessdate={{date|2006-12-16}}}}</ref> The beta version of Windows Live OneCare 1.5 was released in early October 2006 by Microsoft. Version 1.5 was released to manufacturing on {{date|January 3, 2007}} and was made available to the public on {{date|January 30, 2007}}. On {{date|July 4, 2007}}, beta testing started for version 2.0, and the final version was released on {{date|November 16, 2007}}.
