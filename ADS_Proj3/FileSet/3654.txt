Spherical inductance of electron: :<math>L_e = \frac{\mu_0\lambda_0}{2\pi}. \ </math> Characterictic impedance of electron: :<math>\rho_e = \sqrt{\frac{L_e}{C_e}} = \sqrt{\frac{\mu_0}{\epsilon_0}} = \rho_0 = 2\alpha R_H. \ </math> Resonance frequency of electron LC circuit: :<math>\omega_e = \sqrt{\frac{1}{L_eC_e}} = \frac{2\pi c}{\lambda_0} = \frac{m_0c^2}{\hbar}. \ </math> Induced electric flux on electron capacitance: :<math>Q_e = C_eV_e. \ </math> Energy, stored on electron capacitance: :<math>W_{Ce} = \frac{C_eV_e^2}{2} = \frac{Q_e^2}{2C_e} = 2\pi \alpha W_0, \ </math> where <math>W_0 = m_0c^2 - \ </math> is the "rest energy" of electron. So, induced electric flux will be: :<math>Q_e = \sqrt{2\alpha m_0c^2\epsilon_0 \lambda_0} = e. \ </math> Thus, through electron capacitance we have quantized electric flux, equal to the electron charge. Magnetic flux through inductance: :<math>\Phi_e = L_eI_e. \ </math> Magnetic energy, stored on inductance: :<math>W_{Le} = \frac{L_eI_e^2}{2} = \frac{\Phi_e^2}{2L_e} = 2\pi \alpha W_0. \ </math> So, induced magnetic flux will be: :<math>\Phi_e = \sqrt{2\mu_0 hc\alpha} = 2\alpha \Phi_0. \ </math> where <math>\Phi_0 = h/e - \ </math> magnetic flux quantum. Thus, through electron inductance there are no quantization of magnetic flux. ===Bohr atom as LC circuit=== Bohr radius: :<math>a_B = \frac{\lambda_0}{2\pi \alpha} \ </math> where <math>\lambda_0 = \frac{h}{m_0c} - \ </math> Compton wavelength of electron, <math>\alpha - \ </math> fine structure constant. Bohr atomic surface: :<math>S_B = 4\pi a_B^2 \ </math>. Bohr inductance: :<math>L_B = \frac{\mu_0}{\lambda_0}\cdot S_B \ </math>. Bohr capacitance: :<math>C_B = \frac{\epsilon_0}{\lambda_0}\cdot S_B \ </math>. Bohr wave impedance: :<math>\rho_B = \sqrt{\frac{L_B}{C_B}} = \sqrt{\frac{\mu_0}{\epsilon_0}} = \rho_0. \ </math> Bohr angular frequency: :<math>\omega_B = \sqrt{\frac{1}{L_BC_B}} = \frac{\alpha^2}{2}\cdot \frac{m_0c^2}{\hbar} = \frac{2\pi c}{\lambda_B}, \ </math> where <math>\lambda_B = \frac{4\pi a_B}{\alpha }- \ </math> Bohr wavelength for the first energy level. Induced electric flux of the Bohr first energy level: :<math>Q_B = C_BV_B. \ </math> Energy, stored on the Bohr capacitance: :<math>W_CB = \frac{C_BV_B^2}{2} = \frac{Q_B^2}{2C_B} = 2\pi \alpha W_B, \ </math> where <math>W_B = \hbar \omega_B - \ </math> is the Bohr energy. So, induced electric flux will be: :<math>Q_B = \sqrt{2\pi \alpha^3m_0c^2C_B} = e. \ </math> Thus, through the Bohr capacitance we have quatized electric flux, equal to the electron charge. Magnetic flux through the Bohr inductance: :<math>W_LB = \frac{L_BI_B^2}{2} = \frac{\Phi_B^2}{2L_B} = 2\pi \alpha W_0. \ </math> So, induced magnetic flux will be: :<math>\Phi_B = \frac{\pi \alpha^2ec}{\lambda_0}L_B = 2\alpha \Phi_0. \ </math> Thus, through the Bohr inductance there are no quantization of magnetic flux. ===Photon as LC circuit=== Photon "resonant angular frequency": :<math>\omega_w = \sqrt{\frac{1}{L_wC_w}}. \ </math> Photon "wave impedance": :<math>\rho_w = \sqrt{\frac{L_w}{C_w}} = \sqrt{\frac{\mu_0}{\epsilon_0}} = \rho_0. \ </math> Photon "wave inductance": :<math>L_w = \frac{\rho_0}{\omega_w}. \ </math> Photon "wave capacitance": :<math>C_w = \frac{1}{\rho_0\omega_w}. \ </math> Photon "magnetic flux quantum": :<math>\phi_w = L_wI_w = \phi_0 = \frac{h}{e}. \ </math> Photon "wave current": :<math>I_w = \frac{h}{eL_w} = \frac{e\omega_w}{2\alpha}. \ </math> ===Quantum Hall effect as LC circuit=== In the general case 2D- density of states (DOS) in a solid could be defined by the following: :<math>D_{2D} = \frac{m^*}{\pi \hbar^2} \ </math>, where <math>m^* = \xi m_0 - \ </math> current carriers effective mass in a solid, <math>m_0 - \ </math> electron mass, and <math>\xi - \ </math> dimensionless parameter, which considers band structure of a solid.
